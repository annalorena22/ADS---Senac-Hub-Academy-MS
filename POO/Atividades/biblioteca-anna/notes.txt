Model: Cria as classes com seus atributos e as funções CRUD. Representação de tabela do banco de dados.

View: Camada de visualização e interação do usuário.
Controller: Controla os models e executa os métodos.

Model:
Cria as entidades e suas propriedades.
Define as funções que interagem com o banco de dados (CRUD).
Realiza a validação dos dados ou aplica regras de negócios.


Controller:
Recebe as requisições do usuário.
Manipula a lógica de aplicação, usando os Models para fazer operações.
Controla o fluxo de dados entre o Model e a View.
